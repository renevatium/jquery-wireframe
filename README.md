## JQuery Wireframe

Projection renders 3D models (OFF) to a 2D canvas

Built this back in 2013. For the lulz.

### Running

```
npm start
```

- Hold left click to rotate
- Hold right click to transform (move).
- Use scroll wheel to zoom
